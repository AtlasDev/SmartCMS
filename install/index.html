Create The First User

Thank you for installing SmartCMS!

You are now ready to use your brand new CMS! We recomment the following things for you:

  • Delete the install directory from your ftp client. This is not needed, but we do however recomment it. You know, in case we fail you.
  • Add some new text to the home page. The default text is boring, issn't it? You can change it by clicking on the Edit button next to the header, just on the home page, how awsome is that?!
  • Login at the admin backend. Look in the heart of the system!
  • Change the theme of the site. The default theme is awsome, but there are some community made which are awsome too! Take a look at it in the admin backend, at Themes/Install.
  • Add some modules. That's the awsome thing about SmartCMS, Its modulair! You can add awsome things to your site, whitin some mouseclicks. They are both made by our developers (official) and community made (unofficial). You can add things like Forums or Blogs. You can install modules in the admin backend at Modules/Install
